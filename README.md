# dev-orthoboost

Client design previews, served via GitHub Pages at `https://dev-orthoboost.github.io/`.

One folder per mockup — each folder is a self-contained static site and gets its own shareable link:

| Link | Mockup |
| --- | --- |
| [/downtown-landing/](https://dev-orthoboost.github.io/downtown-landing/) | Downtown Orthodontics — dark editorial landing |
| [/downtown-concept-e/](https://dev-orthoboost.github.io/downtown-concept-e/) | Downtown Orthodontics — Concept E (light, pale blue) |
| [/downtown-swiss/](https://dev-orthoboost.github.io/downtown-swiss/) | Downtown Orthodontics — Concept F (Swiss editorial) |

To add a new preview: drop a folder with an `index.html` (+ its assets) at the repo root, add it to the table above and the root `index.html`, commit, push.
